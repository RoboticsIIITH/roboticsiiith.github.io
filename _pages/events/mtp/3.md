### <a id="about-the-program"></a>About the Program

IIIT Hyderabad is identified as one of the Resource Centers (RC) for the MeitY approved project entitled **“Capacity building for human resource development in Unmanned Aircraft System (Drone and related Technology)”**.  

As a part of this project, IIIT Hyderabad organized the **Third Master Trainer Program (MTP)** for the Technology Specific Work Theme **"Allied UAS Technologies"** for five days from **11/03/2024 to 15/03/2024**. :contentReference[oaicite:10]{index=10}  

The program included sessions on simulators for drones, flight controllers, navigation guidance and control (NGC) for UAVs, image processing, and planning.

---

### <a id="program-topics"></a>Program Topics

The five-day program covered: :contentReference[oaicite:11]{index=11}  

- Image Processing (multiple sessions)  
- Ardupilot  
- MUJOCO-based drone simulation (Mujoco I–VIII)  
- Model Predictive Control (MPC) / Path Planning  
- ROS and URDF-ROS  
- Fixed Wing UAV  
- Kinematics
