### <a id="about-the-program"></a>About the Program

The **Fourth Master Trainer Program** for the Technology Specific Work Theme **“Allied UAS Technologies”** focused on **Learning, Vision, and Motion Planning** and was scheduled from **3rd to 7th March 2025** at IIIT Hyderabad.  

The five-day program covered foundations of **computer vision, scientific machine learning, statistical issues in AI/ML, reinforcement learning, motion planning, and applications to UAS**. Tutorial sessions covered implementation aspects of the different methods. :contentReference[oaicite:14]{index=14}  

---

### <a id="program-topics"></a>Program Topics

Key topics covered in the program: :contentReference[oaicite:15]{index=15}  

- Robot Motion Planning  
- Introduction to Image Processing  
- Basics of Reinforcement Learning (RL) and advanced RL lectures  
- Computer Vision  
- Adaptive control using artificial time delay  
- Adaptive control for robots with underactuated and switched dynamics  
- Scientific Machine Learning  
- Non-parametric statistics for AI/ML  
- MAVLink – Drone Communication Protocol  
- Tutorial sessions on implementation and applications
